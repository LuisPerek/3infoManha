Versionamento

O git é um software que permite realizar versões de um projeto. O github desktop é um software que faz a ligação entre vs code, git e github.

O github desktop permite fazer a sincronia entre o repositorio local e o repositorio remoto.

Commit

é um processo de confirmação de uma versao entre, ou seja a cada alteracao é possivel confirmar a alteração

push

este comando envia (sincroniza) as alterações realizadas no repositorio local com o repositorio remoto no github
